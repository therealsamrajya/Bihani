# Bihani 🏥💧

**Bihani** is a mobile health app designed to help users maintain a healthy lifestyle by reminding them to take their medicine, drink water, and follow essential health tips. The app ensures that users stay on track with their wellness routine effortlessly.

## 🚀 Features

- **Medicine Reminders**: Get notified when it's time to take your medicine.
- **Water Intake Alerts**: Stay hydrated with scheduled water reminders.
- **Health Tips**: Receive daily health tips for better well-being.
- **Customizable Alerts**: Set your own reminders based on personal needs.
- **User Dashboard**: View your health schedule and progress in one place.

## 🛠️ Tech Stack

### Frontend:

- React Native (Expo)
- NativeWind (Tailwind CSS for styling)

### Backend:

- Node.js (Express.js)
- MongoDB (Database)

### Authentication:

- JWT-based authentication (Planned)
